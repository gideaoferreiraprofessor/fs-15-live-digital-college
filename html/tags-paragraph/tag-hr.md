# Linha Horizontal

## Tag

```
<hr>
```

## Conceito

Representa uma mudança de assunto ou separação temática dentro do conteúdo.

Visualmente costuma aparecer como uma linha horizontal.

---

## Exemplo

```
<p>Capítulo 1</p><hr><p>Capítulo 2</p>
```

---

## Semântica

Muitos iniciantes acreditam que o `<hr>` serve apenas para desenhar uma linha.

Na verdade, ele indica uma divisão temática no conteúdo.

---

## Exemplo semântico

```
<p>Informações pessoais</p><hr><p>Experiência profissional</p>
```
