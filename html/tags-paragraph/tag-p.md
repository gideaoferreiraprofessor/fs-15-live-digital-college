Parágrafos

## Tag

```
<p></p>
```

## Conceito

Representa um bloco de texto.

Cada parágrafo é tratado como uma unidade independente de conteúdo.

---

## Exemplo

```
<p>HTML é a linguagem responsável pela estrutura das páginas web.</p>
<p>Ela define títulos, parágrafos, imagens, links e diversos outros elementos.</p>
```

---

## Comportamento

Os navegadores adicionam automaticamente um espaço antes e depois dos parágrafos.

```
<p>Primeiro parágrafo.</p>
<p>Segundo parágrafo.</p>
```

---

## SEO

Os mecanismos de busca analisam o conteúdo dos parágrafos para compreender o tema da página.

Textos organizados em parágrafos melhoram a interpretação do conteúdo.

---

## Acessibilidade

Leitores de tela identificam cada parágrafo como uma seção independente de leitura.
